# s8 · hypothesis (C014)

**가설** (s1에서 자란 세 번째 형제 가지): 폴링 `swapRegions`의 통스왑(`replaceChild`)이
열린 카드 노드를 통째 교체해 런타임 상태(네이티브 토글·포커스·리스너·부분 스크롤)를
파괴한다. 열린 노드를 통스왑에서 건너뛰면 보존+실시간이 동시 성립할 것이다.
**JS 마커로 노드 정체성을 헤드리스 관찰** — 내 손 안 계측기로 검증 가능.
s9verify
# s9 · verify (C014)

`hasOpenDetails` 가드로 열린 노드를 통스왑에서 건너뛰게 최소변경. JS 마커로 관찰:
열린 카드는 정체성 보존, 닫힌 영역은 실시간 갱신 — 동시 성립. 참조 133·Go 110

# s10 · analyze (C014) → success (산 잎)

상태 보존의 진짜 단위는 "불린(open)"이 아니라 "노드(정체성)"였다. detKey(C011)는 불린만
복원해 안 나았고, 노드를 건너뛰니 나았다. 최소변경이 "보존 vs 실시간" 트레이드오프를
없앴다. **채택 — 산 잎.** 리젝트 체인(s4·s7)이 이 조준의 계단이었다.
