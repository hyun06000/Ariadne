# 1. 가설 — 실제 원장 전량을 격리 복제본에서 순회 소급 (드라이런)

부모: v3-build/C019-migration-derive-fingerprint (supported — v2 메타→v3 지문 자동 도출).
lineage: v3-build/C017-migration-read-compat (유령 가시성 --report).

## 이전 사이클의 교훈

C019가 한 사이클의 v2 메타→v3 지문 도출 규칙을 세웠다. C019 1순위 제안: **189 전량 순회 소급** — 실제 원장을 순회하며 derive→retro_imprint 배선, `--report` 유령을 0으로. 상현님 지시: **격리 저장소에서 전량 소급 검증**(우리 원장은 안 건드리고 복제본에서 대규모 실측).

## 문제 분할 — 규모가 새 문제를 드러냈다

실제 원장 관찰(1296 커밋):
- gil 원장 커밋 **821**(open 130·step 556·close 135), gil 아닌 커밋 **475**(memory·부트스트랩), 닫힌 사이클 **196**.
- **step/open ≈ 4.3** — 사이클마다 5스텝을 다 채우지 않았다(세션 리밋·조기 종료). C019는 정확히 5스텝을 가정했다.
- **subject 형식 이질**: 표준 `→ N/5`(대부분) + `— N/5 …`(일부) + `gil: release …`(open/step/close 아님) + 부가 텍스트(`open … (수동 시연)`).

C019 도출기는 최신 표준 형식·정확히 5스텝을 가정했다. **규모 확장의 진짜 시험 = 이 이질성에도 견고한가.** 가장 작은 확장 단위: 도출기를 이질 형식에 견고하게 + 전량 순회하며 **못 읽은 것을 정직히 보고**(C017 가시성 — 얼마나 도출됐고 얼마나 유령으로 남나).

## 가설

> **가설(H1)**: 실제 원장의 격리 복제본에서 전량(1296 커밋)을 순회하며 견고화한 도출기로 derive→retro_imprint를 배선하면, gil 원장 커밋 대다수가 v3 지문을 받아 재구성기의 유령 수가 대폭 감소하고, 도출 실패(이질 형식·비-원장 커밋)는 **정직히 보고**되어 남는 유령의 정체가 투명해진다.

- **H1a 전량 순회 무사고:** 복제본에서 821 원장 커밋 순회 소급이 커밋 불변(SHA 안 바뀜)으로 완주 — 우리 원장 무손상(복제본만).
- **H1b 유령 대폭 감소:** 소급 후 재구성기 `--report` 유령 수가 소급 전 대비 대폭 감소(원장 커밋이 지문을 받음).
- **H1c 도출 견고성:** `→ N/5`·`— N/5` 둘 다 파싱, release·비-원장 커밋은 정직히 스킵(오각인 0).
- **H1d 잔여 유령 투명성:** 소급 후 남는 유령(비-원장 memory 커밋·도출 실패)이 정체별로 분류 보고 — 침묵 0.

## 기각 조건

- 복제본 순회가 커밋 SHA를 바꾼다(히스토리 재작성). → H1a 기각.
- 소급 후 유령이 안 줄거나 미미. → H1b 기각.
- 이질 형식(`— N/5`)을 놓치거나 release를 오각인. → H1c 기각.
- 잔여 유령이 정체 불명(침묵). → H1d 기각.
- **우리 실제 원장이 변경된다.** → 치명적 기각(격리 위반).

## 검증 설계 (다음 스텝)

**우리 원장을 격리 복제본(`git clone` 또는 파일 복사)으로** 뜬 뒤 거기서만 작업.
- M1 전량 순회 무사고(H1a): 복제본 821 커밋 소급 완주, 원장 커밋 SHA 전부 불변.
- M2 유령 감소(H1b): 소급 전후 `--report` 유령 수 대조(대폭 감소).
- M3 도출 견고성(H1c): `→`·`—` 둘 다 파싱, release/비-원장 스킵 정확.
- M4 잔여 투명성(H1d): 소급 후 유령을 정체별 분류(비-원장 vs 도출실패) 보고.
- **M5 원장 불변(격리):** 우리 실제 저장소가 이 사이클 동안 안 바뀜(복제본만 각인).

## 정직한 범위 (선긋기)

- **드라이런(격리 복제본)만.** 실제 원장 적용은 이 검증 뒤 별도 결정(상현님). 이 카브는 "전량 소급이 정합·안전한가"만.
- **위상 접합 안 함**(④ 복원). 각 사이클 내 노드 편입만 — 사이클 간 DAG 연결은 이월.
- verdict 다양성은 C019 근사(supported→success) 유지.
- 부분 사이클(5스텝 미만)은 있는 스텝만 도출(없는 스텝 발명 안 함 — C019 "v2에 없던 구조 안 만듦").
