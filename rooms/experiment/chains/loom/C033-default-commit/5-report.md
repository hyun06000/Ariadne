# 5. 결과 보고

## 요약

`gil step`·`gil close`를 깃 저장소에서 **기본 커밋**으로 바꿨다(v1.7.0). --git 없이도 스텝마다 각인, --no-commit으로만 opt-out, 깃 아닌 곳은 조용히 스킵. 두 구현 동일, conformance 26/26 — **채택**. 사용자가 겪은 "사이클 내내 커밋 0" 문제를 기본값 뒤집기로 해결.

## 교훈

1. **규율은 기본값으로 집행될 때 가장 강하다**: §2.1의 "커밋 단위는 스텝"이 문서 규율일 땐 안 지켜졌다. 도구의 기본 동작이 되자 자동으로 지켜진다. 부드러운 강제 = 옳은 기본값.
2. **계약면을 안 건드리는 기본값 변경**: 자동 커밋을 깃 저장소 한정으로 두어 conformance의 비깃 테스트를 보존했다.
3. --git 하위호환 no-op으로 기존 문서·스크립트 무파손.
