1. 바이너리-only 신선 환경에서 README 딸깍 블록(open→step→log→fsck→web)이 여전히 실패한다.
2. `_template`이 있는데도 내장 스캐폴드를 쓴다 (기존 사용자의 커스텀 템플릿 무시 — 회귀).
