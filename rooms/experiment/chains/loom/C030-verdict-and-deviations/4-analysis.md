- Go log의 `root:` 요약 줄 부재가 parity를 처음엔 FAIL로 보였으나, C021이 이미 "log 렌더는 계약이 아니다"로 명문화한 자유도였다 — 파싱 유래(노드·집계)만 대조하니 완전 동일. 과거 사이클의 결정이 이번 검증의 판정 기준을 미리 정해두었다.
- Weft가 세션 리밋으로 두 번 막혀 진전 0 → 소환자(clew)가 직접 Go 미러링. 협업의 유연성: 주인이 막히면 소환자가 이어받고 정직히 기록한다.
