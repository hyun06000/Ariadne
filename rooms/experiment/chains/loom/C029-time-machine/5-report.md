# 5. 결과 보고

## 요약

`gil goto`로 타임머신을 gil 일급 명령으로 만들었다 — 사이클 시점 역행 조회 + `--checkout` 시점 되감기(미커밋 유실 방지) + 분기 재시작 안내. 두 구현 바이트 동일, 검증 전부 통과, conformance 26/26 — **가설 채택**, v1.3.0. 이제 에이전트는 raw 깃 없이 gil 언어로 잘못된 길에서 되돌아간다.

## 교훈

1. **타임머신의 세 동작이 한 명령에**: 조회(어디)·체크아웃(그때로)·분기(다른 길). 사용자의 직관("역행해서 다시 시작")을 gil 언어로 옮겼다.
2. **되감기는 삭제가 아니라 분기다** — 실증: 막다른 길과 새 길이 그래프에 공존. 아리아드네는 잘못을 지우지 않고 건강한 갈림길에서 새 실을 뽑는다. 불변 + 분기 = 안전한 타임머신.
3. **환경 기본값은 반복 함정** — git init 기본 브랜치(master)가 픽스처를 깼다. 세 번째 "테스트를 먼저 의심하라".
