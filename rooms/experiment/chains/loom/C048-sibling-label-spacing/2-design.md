# 2. 실험 설계 — 열 간격을 라벨 폭으로

## 설계 결정

### D1 — `_COL_W`를 라벨 폭 + 오프셋 + 여백으로

라벨은 노드 오른쪽 `x+16`에서 시작해 최대 `label_w=230`px. 겹치지 않으려면 다음 형제(x + _COL_W)가 라벨 끝(x+16+230 = x+246)보다 오른쪽이어야 한다. 여백 14px → **`_COL_W = 260`** (기존 26).

- 선형 체인은 col0만 쓰므로 `col*_COL_W = 0` → **좌표 불변**(하위호환).
- 참조 `_COL_W`, Go `webColW` 둘 다 260으로.
- `label_w`(lane 뒤 여백)는 230 유지 — 마지막 열 라벨 공간.

### D2 — 트레이드오프 수용

세로로 평행한 여러 활성 레인(긴 형제 갈래들)이 있으면 가로가 넓어진다. 그러나 그 노드들의 라벨을 읽으려면 불가피하다 — 상현님이 "간결해진 건 좋다"며 세로 압축을 유지하길 원했으니, 라벨 가독을 위한 가로 확장은 정당한 대가다.
