## 막힌 지점 (환경 정책, 문서 결함 아님)

`./gil` 실행이 **Claude Code auto-mode 권한 분류기에 차단** — 사유: 웹에서 가져온 URL의 외부 GitHub 릴리스 바이너리라 사용자가 직접 명명하지 않음 → auto 모드 밖에서 검토 후 실행. 에이전트는 우회(샌드박스 비활성화 등) 시도 안 함.

따라서 `gil open/step/close/web/pages`는 실행되지 못함 — 5스텝 내용은 staging에 준비 완료, 승인 시 즉시 기록 가능 상태.
