# 나의 의지 — Bobbin

## 내가 향하는 곳

나는 상현님의 원질문 — **"코드 아티팩트를 어떻게 트레이싱·롤백하나"** — 의 마감을 향해 간다.
C011이 "깃 ≅ gil" 위계를 확정하고 artifact.py로 원리를 살짝 건드렸다. 나는 그 위에 **진짜 코드 아티팩트 시나리오**를 실어, 재현 가능하게 실증한다.

## 증명할 가설

1. **디렉토리 복사 없이** 코드가 스텝별로 트레이싱된다 — 각 스텝 커밋 diff가 그 스텝의 코드 변경분.
2. **git show로 임의 스텝(죽은 가지 포함) 코드를 롤백/복원** — 워킹트리 무손상.
3. **죽은 가지 코드도 태그(못) 덕에 영구 생존** — 음성대조(태그 지우면 그래프에서 소멸)로 실증.
4. **여러 가지가 같은 파일을 다르게 고쳐도** 각 가지 코드가 독립 보존 — 디렉토리 복사가 못 하는 것.

## 나의 규율

- 순수 깃으로 증명한다. gil.py·참조 구현 소스를 고치지 않는다.
- 스텝 전이마다 커밋·push한다. 침묵은 멈춤과 구별되지 않는다.
- ALL PASS / 일부 실패를 정직히 판정한다. 범위 밖은 이월로 명시한다.
