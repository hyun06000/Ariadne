# gil wiki — 내비게이션 허브

gil(GIt for Language model) 지식의 진입점이다. LLM이 **필요한 페이지만 골라** 읽도록
쪼개져 있다 — 통째로 읽지 말고, 지금 하려는 일에 맞는 페이지로 바로 가라. 각 페이지는
서로 이어진다.

> 웹에서 이 wiki를 참조한다면 루트의 [llms.txt](../../llms.txt)가 같은 index를 절대 URL로
> 제공한다(사람이 에이전트에게 URL 하나를 건네는 진입점).

## 무에서 시작 — `gil init`

빈 폴더(혹은 gil 세계가 없는 저장소)에서 시작한다면 단 한 명령이면 된다:

```
gil init [--name <이름>]
```

이 한 줄이 ① 대문 커밋(CLAUDE.md 부트스트랩 포인터) ② `refs/gil/global` 초기화
③ 존재의 방(기본 존재 `clew`) ④ refspec 등록·push 를 한다. 그리고 **뷰어가 있으면
관전 서버를 백그라운드로 함께 띄운다** — 브라우저에서 사고 그래프가 자라는 걸 바로 본다.
출력 끝의 `NEXT`가 다음 명령을 프롬프트로 준다. 자세히는 [존재와 기억](existence.md).
