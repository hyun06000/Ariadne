# 목적성 가드

체인·사이클을 열 때 자연어 목적을 못박고, gil이 매 스텝 시작 때 그 목적을 눈앞에 띄워 사고가 표류하지 않게 하는 장치다.

## `--purpose`는 필수

`gil chain`과 `gil open`은 **`--purpose <자연어>`가 필수**다. 목적 없이는 체인·사이클을 열 수 없다.

## gil은 목적을 표시할 뿐

gil은 이후 모든 스텝 시작 때 목적을 띄운다:

```
─ 체인 [demo] 목적: gil 데모 체인
─ 사이클 [c001] 목적: 첫 사이클 데모
─ 지금 하려는 일이 위 목적에 정합하는지 판단하고, 어긋나면 멈춰라.
```

**gil은 정합 여부를 판별하지 않는다 — 목적을 표시하기만 하고, 정합한지는 너(AI)가 판단한다.**

지금 하려는 스텝이 이 목적에서 벗어났다면 멈추고 다시 생각하라. 판단은 AI의 몫이고, 어긋나면 멈추는 것도 AI의 책임이다. 이것이 사고가 목적에서 표류하지 않게 하는 gil의 장치다.
